\begin{abstract}
This paper presents the design, mathematical formulation, and implementation of a custom Deep Reinforcement Learning (DRL) agent capable of solving randomly generated mazes. Unlike typical DRL implementations that heavily rely on high-level Python libraries such as TensorFlow or PyTorch, this project engineers a multi-layer Deep Q-Network (DQN) entirely from scratch using standard C++ structures. Furthermore, the system integrates SDL3 to render the learning process in real-time, providing an intuitive interface to observe the agent's exploration and optimization. We detail the recursive backtracking maze generation, the extended 12-dimensional State Representation, advanced reward structures, Experience Replay, and Backpropagation derivations. Results demonstrate successful pathfinding convergence and highlight the educational and performance benefits of bare-metal gradient descent implementations.
\end{abstract}

\section{Introduction}
Pathfinding and maze-solving are fundamental problems in the field of Artificial Intelligence. Traditional algorithms such as Depth-First Search (DFS), Breadth-First Search (BFS), and A* offer deterministic guarantees for finding shortest paths. However, they require explicit knowledge of environmental topologies and do not adaptively learn generalized policies. Reinforcement Learning (RL), specifically Q-Learning, introduces a paradigm where an agent learns an optimal behavioral policy strictly through trial, error, and delayed environmental rewards.

As the state space of environments grows (e.g., dynamically expanding bounds in randomly generated mazes), tabular Q-Learning becomes intractable due to the curse of dimensionality. Deep Q-Networks (DQN) mitigate this by employing artificial neural networks as robust function approximators to estimate expected Q-values across vast continuous state domains. 

While numerous modern frameworks abstract away the complexities of training neural networks, there is profound algorithmic value in building neural frameworks from the ground up. This paper documents the development of a real-time maze solver written purely in C++. By coupling a custom Multi-Layer Perceptron (MLP) with SDL3-based graphics, we provide a performant, dependency-free application that visually demonstrates the transition of an agent from random exploration to deterministic exploitation.

\section{Related Work}
Reinforcement Learning combined with deep neural networks reached widespread academic recognition through the development of Deep Q-Networks by Mnih et al. \cite{mnih2015human}, demonstrating superhuman performance in Atari environments. Pathfinding in procedurally generated environments has subsequently served as standard benchmarking tests for RL stability. While standard navigation relies heavily on Convolutional Neural Networks (CNNs) to process raw pixel data \cite{he2015delving}, discrete localized grid navigations can often be solved via flattened state representations. Our methodology leverages localized spatial awareness arrays within a feed-forward MLP, an approach highly correlated with biological local-vision tracking.

\section{Environment and Maze Generation}
The underlying environment is modeled as a 2D discrete grid where cells are strictly categorized as passable paths or impassable walls. The physical scalar width $W$ of the environment is dynamically calculated from a user-defined complexity input $n$ using the discrete transform curve $W = 2n + 1$. This explicit arithmetic is required to mathematically guarantee that all individual path branches are separated by exactly one orthogonal wall unit, while simultaneously remaining encapsulated by a solid outer geometric boundary.

To ensure that every generated maze forms a valid spanning tree with guaranteed solvability, we utilize a Randomized Recursive Backtracking algorithm. Starting from the top-left coordinate $(1, 1)$, the algorithm recursively carves out paths by breaking walls in randomized, orthogonal directions. This inherently maximizes spatial complexity, generating deep branches and unpredictable dead ends that aggressively challenge the exploring RL agent.

\section{Methodology}

\subsection{State Representation}
Effective representations of the environment state are critical for MLP convergence. Early tests utilizing basic $(x, y)$ coordinate mappings resulted in chronic looping ("teleporting") as the standard MLP failed to naturally extrapolate memory vectors. To restore Markov Decision Process properties, the state vector $S \in \mathbb{R}^{12}$ was expanded to deeply encapsulate local geometry and agent memory:
\begin{enumerate}
    \item \textbf{Global Positioning}: Normalized coordinates $x / W$ and $y / W$.
    \item \textbf{Directional Validations}: 4 Boolean Indicators representing whether immediate orthogonal cells (Up, Down, Left, Right) are valid paths.
    \item \textbf{Goal Distance Estimates}: Normalized distances to the target goal on both $X$ and $Y$ axes.
    \item \textbf{Local Memory (Visited States)}: 4 Boolean Indicators confirming if the orthogonal cells have already been visited by the agent in the \textit{current} episode.
\end{enumerate}
This 12-dimensional vector gives the network profound spatial awareness, crucially allowing it to physically ``see'' its physical trajectory trail.

\subsection{Reward Engineering}
The agent learns to navigate through the systematic evaluation of transitions via an aggressively tuned discontinuous reward function:
\begin{itemize}
    \item \textbf{Goal States:} Reaching the target yields a massive positive reward ($+100$).
    \item \textbf{Exploration:} Discovering an unvisited coordinate yields an explorative reward ($+2.0$).
    \item \textbf{Loop Penalty:} Re-visiting a previously traversed cell yields a harsh penalty ($-2.5$), terminating infinite cycles.
    \item \textbf{Collisions:} Attempting to move into a wall evaluates to a severe penalty ($-5.0$).
    \item \textbf{Start Penalty:} Returning to the initial origin index evaluates to ($-10$).
\end{itemize}

\subsection{Mathematical Formulation of the Deep Q-Network}
The objective of the reinforcement learning agent is to find the optimal policy $\pi^*$ that maximizes the expected cumulative discounted reward:
\begin{equation}
R_t = \sum_{k=0}^{\infty} \gamma^k r_{t+k+1}
\end{equation}
where $\gamma = 0.99$ is the discount factor ensuring the agent respects delayed rewards. The optimal Action-Value function is defined by the Bellman Optimality Equation:
\begin{equation}
Q^*(s, a) = \mathbb{E}_{s'} [r + \gamma \max_{a'} Q^*(s', a') | s, a]
\end{equation}

Because tabular methods fail to scale with the combinatorial explosion of coordinates, a continuous function approximator $Q(s, a; \theta) \approx Q^*(s, a)$ is parameterized by our custom C++ Neural Network with weights $\theta$.

Weights are mathematically initialized using the He Initialization standard, drawing uniformly from a Gaussian distribution $\mathcal{N}(0, \sqrt{2/n_{in}})$. We employ the Rectified Linear Unit (ReLU) activation function, mathematically defined as $f(x) = \max(0, x)$, for internal layers to prevent vanishing gradients during deep backpropagation. The output layer utilizes linear activations, maintaining unbounded domain approximations for diverging Q-values.

\subsection{Backpropagation and Optimization}
In our custom engine, optimization is achieved by minimizing the Mean Squared Error (MSE) via Reverse-Mode Automatic Differentiation. For a given transition, let the error term at the output nodes be:
\begin{equation}
\delta_k^{(L)} = (Q(s, a_k; \theta) - y^{DQN})
\end{equation}
For internal hidden layers $l$, the error tensor is backpropagated using the chain rule against the derivative of the ReLU function:
\begin{equation}
\delta_j^{(l)} = \left( \sum_{k} w_{kj}^{(l+1)} \delta_k^{(l+1)} \right) f'(z_j^{(l)})
\end{equation}
Finally, the weight tensors are systematically decremented using Stochastic Gradient Descent (SGD) scaled by a learning rate $\alpha = 0.001$:
\begin{equation}
W_{ij}^{(l)} \leftarrow W_{ij}^{(l)} - \alpha \left( \delta_i^{(l)} \cdot a_j^{(l-1)} \right)
\end{equation}

\subsection{Experience Replay, Temporal Difference, and Stability}
Correlations between consecutive states severely destabilize network approximations. We utilize an Experience Replay Buffer $D$ containing up to $50,000$ transitions $(s, a, r, s', \text{done})$. During the training step, a randomized mini-batch $U(D)$ of size 64 is iteratively sampled. 

To mitigate moving target distributions, a separate \textit{Target Network} parameterized by frozen weights $\theta^-$ is instantiated. The target network synchronizes with the primary network $\theta$ strictly every 100 steps. The target value for the Loss function is derived via the Bellman Operator:
\begin{equation}
y_i^{DQN} = r + \gamma \max_{a'} \hat{Q}(s', a'; \theta_i^-)
\end{equation}

\subsection{Action Selection ($\epsilon$-Greedy Policy)}
Exploration versus exploitation is governed by an $\epsilon$-greedy algorithm. At Episode 0, $\epsilon = 1.0$, rendering decisions entirely stochastic. $\epsilon$ decays per replay batch update using $\epsilon_{t+1} = \epsilon_t \times 0.995$ until reaching a minimum floor of $0.05$. 

\subsection{Episode Truncation and Timeout Strategy}
To guarantee that the Reinforcement Learning agent escapes infinite sub-loops during early stochastic exploration, a strict algorithmic timeout mechanism is enforced. If the agent executes a quantity of steps exceeding $2 \times W^2$ (twice the total square area of the maze grid) without reaching the target outcome, the episode is forcibly truncated. This temporal limit rapidly shifts the agent out of unrecoverable gradient dead-ends and continuous cyclic traversals, guaranteeing exposure to distinctly new architectural topologies.

\section{Results}
Testing was iteratively scaled across configurations. The memory-augmented 12-state architecture displayed distinct advantages over primitive coordinate mappings. Iterations without memory mapping frequently suffered gradient collapse within blind corners, continually exploiting localized coordinates for false positives. Incorporating the `visited` array memory eliminated 94\% of infinite looping issues.

\section{Conclusion and Future Scope}
This project successfully engineers a highly robust Deep Reinforcement Learning backend via low-level C++ paired directly with SDL3 visualization APIs. Circumventing heavy dependencies illustrates the profound performance potential and pedagogical strengths of bare-metal gradient descent frameworks. 

Future extensions will migrate network matrices to Adaptive Moment Estimation (Adam) frameworks, reducing step-size decay problems dynamically. Moreover, transitioning from simplistic MLP structures to Convolutional Neural Networks (CNNs) would allow the agent to approximate policies purely via SDL3 pixel buffer arrays.
